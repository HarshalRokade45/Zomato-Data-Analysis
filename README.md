# Zomato Data Analysis using Tableau

1.Project Overview

   This project analyzes Zomato restaurant data using Tableau to extract meaningful insights. The dashboard helps understand customer preferences, restaurant         distribution, pricing trends, and online delivery patterns.

2. Objectives

   * Analyze restaurant ratings and votes
   * Identify city-wise restaurant distribution
   * Study online delivery availability
   * Understand cost variations across cities

3. Dataset

   The dataset contains information such as:
      * City
      * Cuisines
      * Votes
      * Aggregate Rating
     * Average Cost for Two
      * Online Delivery , etc.

4. Visualizations

   - Bar Chart → Restaurants per City
   - Scatter Plot → Votes vs Rating
   - Pie Chart → Online Delivery Distribution
   -Line Chart → Average Cost by City

5. Dashboard Preview
   
   https://public.tableau.com/app/profile/harshal.rokade/viz/ZomatoDashboard_17770298445860/Dashboard1?publish=yes

 6. Key Insights

   * Restaurants with higher votes tend to have better ratings
   * Major cities have a higher number of restaurants
   * Online delivery is widely available
   * Cost varies significantly across locations

7.Tools Used

   Tableau (Data Visualization)


8.Conclusion

This project demonstrates how data visualization can transform raw data into actionable insights for better decision-making in the food industry.
